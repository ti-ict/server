# Overzicht

De infra draait op twee Dell PowerEdge R620 servers met Proxmox VE.

## Topologie

- **2x PowerEdge R620**: hoofdservers die Proxmox VE draaien (zie [Dell PowerEdge R620](./hardware/poweredge-r620))
- **Oude server (OPNSense)**: router/firewall
- **Switch**: Dell N1548P (48 poorten, Gigabit, VLAN-capable, PoE) (zie [Dell Networking N1548P](./hardware/dell-n1548p))
- **Netwerk**: VLANs voor servers, VMs en PCs (zie [Netwerk](./network))
